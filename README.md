# 스마트 알약 디스펜서 (Smart Pill Dispenser)

> **"고령층의 복약 순응도를 높이는 자동 배출 및 긴급 알림 시스템"**
> 앱 설정 시간에 맞춰 알약을 자동 배출하고, 초음파 센서로 복용 여부를 확인하여 미복용 시 보호자에게 알림을 전송합니다.

## 1. 프로젝트 개요
* **배경**: 고령화 사회에서 독거 노인의 약 복용 누락 문제를 해결하기 위해 기획되었습니다.
* **목적**: 정해진 시간에 알약을 자동 배출하고, 미복용 시 다중 알람(LCD, 부저, 앱)을 통해 안전한 복약 환경을 제공합니다.
* **기간**: 2023.10 ~ 2023.12 (약 2개월)
* **성과**: **2023 한국정보기술학회 대학생논문경진대회 금상 수상**

## 2. 핵심 기능

| 기능 | 상세 설명 |
| :--- | :--- |
| **자동 배출 시스템** | 앱에서 설정한 시간에 스텝모터(약통 정렬)와 서보모터(배출)가 순차 작동 |
| **복용 여부 감지** | 초음파 센서로 배출구 앞 300mm 이내 물체를 감지하여 실제 복용 여부 판단 |

| **안드로이드 연동** | Java 기반 앱을 통해 복약 시간 설정 및 실시간 기기 상태 모니터링 |

## 3. 시스템 아키텍처

### **Data Flow**
1. **입력**: Android 앱(알람 설정) → 블루투스(HC-05) → **ATmega128**
2. **처리**: 초음파 센서 실시간 거리 측정 및 상태 머신(State Machine) 제어
3. **출력**: 스텝모터(60° 회전) / 서보모터(배출) / LCD(상태 표시) / 부저(경고음)

## 6. 주요 코드 구현

### **초음파 기반 거리 측정 (measure_distance)**
Trigger 핀에 10µs 펄스를 인가한 후 Echo 핀의 응답 시간을 카운트하여 mm 단위로 거리를 계산합니다.

```c
uint16_t measure_distance(void) {
    uint32_t count = 0;
    PORTB |= (1 << TRIG); // Trigger High
    _delay_us(10);
    PORTB &= ~(1 << TRIG); // Trigger Low

    while (!(PINB & (1 << ECHO))); // Echo핀이 High가 될 때까지 대기
    while (PINB & (1 << ECHO)) {   // Echo핀이 High인 동안 카운트
        _delay_us(1);
        count++;
        if (count > 60000) break;   // 타임아웃 처리
    }
    // 거리 계산 로직 (음속 기반)
    return (uint16_t)((count * SOUND_VELOCITY) / (2UL * 10000UL));
}

## 7. 주요 기술적 해결 (Troubleshooting)

### **a. 멀티 디바이스 타이밍 간섭 해결**
* **문제**: 모터 동작, 초음파 센서 읽기, LCD 업데이트가 동시에 발생할 때 타이밍 충돌로 시스템이 멈추거나 센서 데이터가 튀는 현상 발생.
* **원인**: 각 모듈에서 사용하는 `_delay_ms` 함수가 CPU 점유를 독점하여 다른 프로세스의 실행을 방해함.
* **해결**: 
  * **상태 머신(State Machine)** 구조를 도입하여 시스템 동작 단계를 세분화.
  * 불필요한 딜레이를 제거하고 비차단식(Non-blocking) 설계를 통해 각 기기의 동작 순서를 명확히 분리하여 실시간 응답성 확보.

### **b. 하드웨어 정밀도 보정**
* **문제**: 초기 아크릴 케이스 가공 시 모터 축과 약통 칸의 물리적 정렬이 맞지 않아 알약이 걸리거나 배출되지 않는 문제 발생.
* **해결**: 
  * **물리적 수정**: 납땜기를 활용한 열 가공으로 기구부 오차를 직접 수정.
  * **소프트웨어 보정**: 스텝모터의 회전 수(`Step`)를 소프트웨어적으로 미세 조정하는 로직을 추가하여 정밀한 정렬 구현.
